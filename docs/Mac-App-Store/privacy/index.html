Privacy Policy

Keep monitoring local.

Core Monitor for the Mac App Store does not require an account, run analytics SDKs, or build remote profiles.

Effective April 19, 2026. This policy applies to the Mac App Store edition of Core Monitor.

Back to App Store page Support Back to Core-Monitor

No account

No sign-in, no subscription gate, no personal profile.

The app operates locally on the Mac and uses system status data only to render the interface.

Location is optional

Requested only when weather is enabled.

If you do not turn on weather, the app does not ask for location access.

Network use

WeatherKit and Apple Weather attribution only.

No analytics SDKs, advertising SDKs, tracking beacons, or telemetry pipeline are included in this edition.

Data Collection

What the app reads.

Core Monitor reads limited device status locally so the Mac App Store edition can show current system conditions.

  • CPU activity and per-core load
  • Memory usage and pressure
  • Battery and power-source status
  • Thermal state and warning level
  • Network throughput
  • Startup disk usage
  • Uptime and load averages
  • Local weather when you grant location access

Not Collected

What the app does not do.

This edition is designed to avoid tracking infrastructure and remote user profiling.

  • No account creation or sign-in requirement
  • No analytics SDKs
  • No advertising SDKs
  • No tracking or telemetry pipeline
  • No remote profile building
  • No collection of personal profiles

Permissions

  • Location is requested only when you enable weather.
  • The app does not request camera, microphone, contacts, photos, accessibility, input monitoring, automation, or full disk access.

Network and Support

  • The network is used only for WeatherKit data and the Apple Weather attribution assets required while weather is active.
  • Privacy or support questions can be sent to coremonitor.app@gmail.com or through the public support page.

Retention and Deletion

  • This edition does not require an account and does not store personal data on Core Monitor servers.
  • You can revoke location access in macOS System Settings or stop weather use in the app at any time.

Contact

  • Support email: coremonitor.app@gmail.com
  • Public support page: Core Monitor Support
